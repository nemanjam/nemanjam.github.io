## 🔧 配置指南

### 修改精选文章

编辑 `src/pages/index.astro`，修改 `featuredPosts` 数组：

```typescript
const featuredPosts = [
  {
    title: '你的文章标题',
    subtitle: '精选文章',
    category: '分类名称',
    tags: ['标签1', '标签2', '标签3'],
    heroImage: '/path/to/image.jpg',  // 放在 public/ 目录
    link: '/blog/your-post',
    bgColor: '#4A5568',  // 自定义背景色
  },
  // 最多添加3篇
];
```

### 修改特点卡片

编辑 `featureCards` 数组：

```typescript
const featureCards = [
  {
    title: '卡片标题',
    description: '卡片描述文字...',
    icon: 'mdi:icon-name',  // 或 'shopify'（你添加的自定义图标）
    link: '/your-link',
    bgColor: '#5F9E6C',  // 自定义背景色
  },
  // 可以添加更多卡片
];
```

### 修改最新文章

目前使用示例数据。要使用真实文章，需要从 Content Collection 获取：

```typescript
import { getCollection } from 'astro:content';

const allPosts = await getCollection('post', ({ data }) => {
  return !data.draft;
});

// 按发布日期排序
const sortedPosts = allPosts.sort((a, b) =>
  b.data.publishDate.valueOf() - a.data.publishDate.valueOf()
);

const latestPosts = sortedPosts.slice(0, 5).map(post => ({
  title: post.data.title,
  description: post.data.description,
  image: post.data.heroImage,  // 需要处理图片路径
  tags: post.data.tags,
  link: `/blog/${post.slug}`,
}));
```

## 📸 准备图片

2. **文章缩略图**
   - `post-1.jpg` 到 `post-5.jpg`
   - 建议尺寸：800x450px（16:9比例）
